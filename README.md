# The Preacher's Tool

> *"Preach the word; be ready in season and out of season."*
> — 2 Timothy 4:2 (NASB 1995)

A sermon, Bible study, workbook, and print-preparation workflow plugin for Obsidian — built for preachers, teachers, and biblical educators who take the Word seriously.

---

## Purpose

> *"Study to show yourself approved to God, a workman who does not need to be ashamed, rightly dividing the word of truth."*
> — 2 Timothy 2:15 (NASB 1995)

The Preacher's Tool provides a complete writing and preparation environment inside Obsidian for:

- Full sermon manuscripts and live preaching outlines
- Inductive, verse-by-verse, topical, expository, and doctrinal Bible studies
- Workbook modules, study guides, and fill-in-the-blank lesson materials
- Print-ready PDF export profiles

---

## Requirements

- Obsidian 1.5.0 or higher
- [Style Settings plugin](https://github.com/mgmeyers/obsidian-style-settings) — required for theme customization

---

## Installation (Manual)

1. Download or clone this repository
2. Copy the `the-preachers-tool` folder into your vault's `.obsidian/plugins/` directory
3. In Obsidian: **Settings → Community Plugins → Enable The Preacher's Tool**
4. Install the **The Preacher's Tool Theme** via **Settings → Appearance → Themes**

---

## Plugin Settings

Navigate to **Settings → The Preacher's Tool** to configure your defaults:

| Setting | Description |
|---|---|
| Default Speaker | Your name as it appears on sermon headers |
| Default Location | Your church or congregation name |
| Default Bible Version | Bible translation used in templates (e.g. NASB 1995) |
| Default Sermon Palette | Color palette applied to sermon documents |
| Default Study Palette | Color palette applied to study documents |
| Default Workbook Palette | Color palette applied to workbook documents |

## Theme — Style Settings

With the **Style Settings** plugin installed and **The Preacher's Tool Theme** active, navigate to **Settings → Style Settings** to customize:

- **Body Font** — System, Serif, Sans-Serif, Palatino, Lexicon, Minion Pro
- **Heading Font** — Georgia, Times New Roman, Garamond, Lexicon, Minion Pro
- **Base Font Size** — Normal (16px), Large (18px), Extra Large (20px), Pulpit (22px)
- **Line Height** — Compact, Normal, Relaxed, Wide
- **Title & Heading Alignment** — Left, Center, Right
- **Color Palette** — 70 named palettes for document theming
- **Hide Frontmatter in Reading View** — Toggle
- **Hide Link Decorations in Print** — Toggle

---

## Print & Export Workflow

> *"Let all things be done decently and in order."*
> — 1 Corinthians 14:40 (NASB 1995)

1. Write your sermon, study, or workbook using the TPT templates
2. Apply the appropriate print profile via the Command Palette
3. Switch to **Reading View**
4. Use **Cmd+P → Export to PDF** to generate your print-ready document
